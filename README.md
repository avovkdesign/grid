# grid.css

Simple CSS grid

## Use

```
<div class="grid">
    
    <!-- row 1 -->
    <div class="col12">Full width block</div>
    
    <!-- row 2 -->
    <div class="col4 newrow">4/12 width and clear left margin with 'newrow'</div>
    <div class="col4">4/12 width</div>
    <div class="col4">4/12 width</div>

</div>
```
